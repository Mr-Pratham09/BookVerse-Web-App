# BookVerse

This project is a modern Book Store UI (BookVerse) built using React.js and Tailwind CSS. It features a visually rich, responsive interface for browsing books, exploring recommendations, and interacting with UI components like sliders, cards, and modals. The project focuses on frontend design, component reusability, and modern UI/UX practices.

## Built With

- React.js
- Vite
- Tailwind CSS
- AOS (Animations)
- React Slick (Carousel)

## Key Features
Hero Section with Dynamic Slider — Interactive book showcase with switching content
Responsive Navbar — Clean navigation with dropdown and search UI
Order Popup Modal — Simple UI form for ordering books
Books Grid Section — Displays book cards with ratings and author info
Services Section — Highlighted book categories with hover effects
Testimonials Slider — Customer feedback carousel
App Download Banner — Mobile app promotion section
Fully Responsive Design — Optimized for all screen sizes
Modern UI/UX — Smooth animations and clean layout

## Note
This is a learning project built for frontend development practice and UI improvement. The main focus of this project was to strengthen skills in React.js, component structuring, and modern UI design using Tailwind CSS. It helped in gaining hands-on experience with building responsive layouts, reusable components, and interactive user interfaces.

## Installation

To run this project locally:

1. Clone the Repository:
```bash
git clone git@github.com:Mr-Pratham09/BookVerse-Web-App.git
```

2. Navigate to the project folder:
```bash
cd my-app
```

3. Install the dependencies:
```bash
npm install
```

4. Start the development server:
```bash
npm run dev
```

Now, you can open your browser and go to http://localhost:5173 to view the app!
